Система модерации с защитой администрации

## 📦 Установка
1. Скачайте `ZenexReport.jar`
2. Поместите в папку `plugins`
3. Перезапустите сервер

## 🎯 Команды
| Команда | Описание | Право |
|---------|----------|-------|
| `/report <игрок> <причина>` | Пожаловаться | `zenexreport.report` |
| `/punish <игрок> <ban\|mute\|kick\|warn> <причина>` | Наказать | `zenexreport.moderator` |
| `/history <игрок>` | История наказаний | `zenexreport.moderator` |
| `/zp reload` | Перезагрузка | `zenexreport.admin` |

## 🔐 Права
| Право | Описание |
|-------|----------|
| `zenexreport.report` | Право жаловаться |
| `zenexreport.moderator` | Права модератора |
| `zenexreport.admin` | Административные права |
| `zenexreport.bypass` | Обход наказаний |

## 🔐 Защита администрации
4-значный код при входе
